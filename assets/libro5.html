4. ¿Qué es un sistema de ecuaciones lineales?

En los capítulos anteriores, hemos trabajado con intersecciones entre distintos objetos.

En lo que sigue intentaremos saber qué es un sistema de ecuaciones lineales y qué es una solución del sistema.

5. ¿Qué es el conjunto solución de un sistema de ecuaciones lineales?

Una solución del sistema es un vector de n componentes tal que cada una de sus componentes reemplazadas en las variables satisface todas las igualdades del sistema.

El conjunto solución de un sistema de ecuaciones lineales es el conjunto de todos los vectores que son solución.

6. ¿Cómo hallar el conjunto solución de un sistema de ecuaciones lineales?
